## About The Project

The "Build The Earth" (BTE) project is a global collaborative initiative that aims to recreate the entire Earth in Minecraft at a 1:1 scale. The project spans thousands of contributors worldwide and relies on external tools, geographic data. However, due to the complexity and scale of the project, manual recreation of landscapes and cityscapes remains a time-consuming task.

Within BTE Romania + Moldova, Google Earth is the main source of geospatial data thanks to its high-fidelity 3D Photogrammetry data for several regions including Bucharest, Cluj-Napoca, Timișoara and Oradea. Currently, the virtual building process is manual in its entirety relying on a lengthy process of measurements and planning to match the real-life proportions and position of buildings and other features. Therefore, an automated solution capable of seamlessly integrating Google Earth’s 3D mesh data into Minecraft would greatly accelerate the building process and make the project more accessible to newcomers.

BTE Voxelizer is intended to help speed up the building process and make it more accessible to newcomers. The tool creates a block-based representation of a user selected area from the Google Earth 3D mesh.
